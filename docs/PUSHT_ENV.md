# PUSHT_ENV 训练环境说明

## 一、 位姿分布与训练边界

### 1. 核心随机化（默认 Episode 级别）
每个 Episode 默认随机化 3 个自由度（FoV）：
* **Agent 起始位置**：$x, y \in [50, 450]$
* **T-block 起始位置**：$x, y \in [100, 400]$
* **T-block 初始角度**：$\theta \in [-2\pi, 2\pi]$
* **注意**：目前代码尚未实现 Agent 与 Block 的初始重叠检测约束（TODO）。

### 2. 目标状态 (Goal State)
* **默认状态**：目标位置通常固定在屏幕中心 `[256, 256]`，角度为 $\pi/4$（视觉 Overlay 呈现）。
* **采样机制**：若未外部指定目标，系统会独立于初始位姿，再次在上述边界内采样一组 Agent 和 Block 的位姿作为目标状态。

### 3. 空间物理边界
* **墙体边界**：场景物理墙范围约为 $[5, 506] \times [5, 506]$。

---

## 二、 成功标准与奖励函数

### 1. 成功终止条件 (Terminated)
必须同时满足以下两个几何精度要求：
* **位置偏差**：$\| \text{goal\_state}[:4] - \text{state}[:4] \|_2 < 20$ （需同时包含 Agent 和 Block 的位置）。
* **角度偏差**：$\Delta\theta < \pi/9$ (约 20°)。

### 2. 奖励计算 (Reward)
* **计算方式**：采用负欧几里得距离，即 $R = -\| \text{goal\_state} - \text{state} \|_2$。

### 3. 步数与评测统计
* **步数上限**：由 `TimeLimit` 容器控制，默认配置为 **100 步**。
* **截断机制**：环境内部不进行截断 (`truncated=False`)，由外部包装器处理。
* **成功定义**：仅当触发 `terminated`（满足位姿精度）时计为成功；若达到最大步数被强制截断 (`truncated`)，即使距离很近也计为失败。

---

## 三、 固定参数与随机化范围

### 1. 静态物理/几何属性（默认保持不变）
* **物理常数**：无重力 $(0, 0)$、无阻尼 (`damping=0`)、时间步长 $dt=0.01$。
* **动力学系数**：控制增益 $k_p=100, k_v=20$，动作缩放 $action\_scale=100$，各物体摩擦力 $friction=1$。
* **外观规格**：
    * **Block**：T 型形状，缩放比例 40，颜色固定。
    * **Agent**：圆形，缩放比例 40，初始角度为 0，初始速度为 0。
    * **环境**：白色背景。
