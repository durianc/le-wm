# Two Room Environment 详解
### 坐标系与环境设置

- **画布大小**：224 × 224 像素
- **有效区域**：`[BORDER_SIZE, IMG_SIZE - BORDER_SIZE - 1]` = `[14, 209]`（x 和 y 轴）

### 固定结构（不随 episode 变化）

| 参数                | 值          | 说明                                      |
|---------------------|-------------|-------------------------------------------|
| `WALL_CENTER`       | 112         | 垂直墙的 x 坐标（固定不动）               |
| `WALL_WIDTH_DEFAULT`| 10          | 墙宽度，默认 10px，实际占 [107, 117]      |
| `wall_axis`         | 1（垂直）   | 默认垂直墙，将画布左右分隔                |

### 每 episode 随机的参数（默认 variation）

```python
DEFAULT_VARIATIONS = ('agent.position', 'target.position')
```

| 参数              | init_value（采样中心） | 采样范围                                      | 是否随机 |
|-------------------|------------------------|-----------------------------------------------|----------|
| `agent.position`  | [60.0, 112.0]          | [14, 209] × [14, 209]，需过滤墙内             | 是       |
| `target.position` | [164.0, 112.0]         | [14, 209] × [14, 209]                         | 是       |
| `door.position[0]`| 49（固定）             | [0, 223]                                      | 否       |
| `door.size[0]`    | 14（固定）             | 门半径                                        | 否       |
| `door.number`     | 1                      | -                                             | 否       |

### 关键结论

- 训练集中**墙位置固定**在 `x = 112`
- **门位置固定**在 `y ≈ 49`（半径 14px），实际门覆盖范围 `y ∈ [35, 63]`
- Agent 起点在**左室**（`x < 112`）附近采样
- 目标点在**右室**（`x > 112`）采样

### 数据集统计（来自配置注释）

- Agent x 范围：`[14, 209]`
- 87.4% 的 episodes 会跨越 `x = 112`
- Agent 在靠近墙时（`x ∈ [100, 124]`）的 y 坐标：
  - 均值 ≈ 53，标准差 ≈ 7
  - p10 = 42，p90 = 62
- → 实际过墙的 y 质心 ≈ **53–54**（与 `door_y_fact=54` 高度吻合）

### 成功终止条件

```python
terminated = dist < 16.0   # 欧氏距离 < 16px
```
